About me

Hello, my name is Nabeel Hussain and I'm a computer science student at University of Maryland, University College. I currently work for a startup company based out of College Park, MD called TalkLocal, where I am the Operations Manager for the customer service team. I am a self taught web designer and constantly try to learn new things everyday with the hopes of becoming a full stack web developer in the near future. I also hold a degree in Economics from the University of Maryland, College Park where I graduated back in 2012.

Wikipedia Viewer

Displays Wikipedia articles for any search queries using an API.

This app uses the Wikipedia API to retrieve and display the top 10 search result articles for a given search query. The API is retrieved using JSON and Jquery.

Visit website

Local Weather App

Displays the users current outside weather information using an API.

This app uses two API's to determine your location and then find the weather in your current area. To find the location, I used an API I found at http://ipinfo.io/developers in which I was able to retrieve the city and geographical coordinates. I then used those coordinates as an input parameter in the weather API I found at https://openweathermap.org/api. The API's are retrieved using JSON and jQuery.

Visit website

Random Quote Generator

Displays random quotes to the user using an API

This is a random quote generator that uses an API I found online from http://quotes.stormconsultancy.co.uk/api in order to generate random programming quotes. It also incorporates twitter social media integration to allow users to tweet out any quote that they like.
